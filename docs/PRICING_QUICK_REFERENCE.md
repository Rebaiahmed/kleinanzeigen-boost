- ✅ **Analytics** — basic views/clicks (not heatmaps)

## 📊 Key Metrics to Track

```
Monthly Recurring Revenue (MRR):
= (Starter users × €4.99) + (Pro users × €9.99)

Customer Acquisition Cost (CAC):
= marketing spend / new paid customers

Lifetime Value (LTV):
= €4.99 × avg Starter months + €9.99 × avg Pro months

Payback period:
= CAC / LTV

Churn rate (aim: <5%/month):
= (customers lost this month) / (starting customers)
```

## ⚠️ Pricing Risks & Mitigations

| Risk | Mitigation |
|------|-----------|
| Too expensive? | Start at €4.99. If no traction by month 6, drop to €3.99 |
| Too cheap? | After 6 months at 500+ users, raise to €5.99 / €11.99 |
| Free tier costs too much | Enforce tighter limits (10 photos instead of 15) |
| Users churn after trial | Improve onboarding; track what keeps users active |
| Pro tier neglected | Add more power-user features (bulk actions, API, webhooks) |
